Internet and Web Fundamentals

Internet

What is Internet?

The Internet is a worldwide network of interconnected computers and devices that communicate with each other using standard communication protocols such as TCP/IP (Transmission Control Protocol/Internet Protocol). It enables users to share information, access websites, send emails, stream videos, communicate instantly, and use various online services from anywhere in the world.

Characteristics

  1. Worldwide connectivity
  2. Fast communication
  3. Resource sharing
  4. Open standards
  5. Scalable network
  6. Supports multimedia

Advantages and Disadvantages

Advantages Disadvantages
Global Connectivity: People can stay in touch across thousands of miles instantly using messaging apps, video calls, and social networks. Misinformation & Fake News: Unverified claims and unedited content can spread quickly, requiring users to critically evaluate what they read online.
Abundant Information & Education: The web acts as a massive encyclopedia , offering online courses, research databases, and tutorials on almost any imaginable subject Addiction & Time Wastage: Excessive screen time and digital distractions can easily lead to procrastination and lowered productivity.

History of the Internet

  1. 1960s – The Beginning

    The Internet began as a research project funded by the U.S. Department of Defense. A network called ARPANET (Advanced Research Projects Agency Network) was developed to connect computers at different research institutions. The first successful communication between two computers occurred in 1969.

  2. 1970s – Development of TCP/IP

    Researchers developed the TCP/IP communication protocol, which became the standard method for transferring data between computers. This allowed different networks to communicate with one another.

  3. 1983 – Birth of the Modern Internet

    On January 1, 1983, ARPANET officially adopted TCP/IP. This milestone is considered the birth of the modern Internet.

  4. 1989–1991 – World Wide Web

    Tim Berners-Lee invented the World Wide Web (WWW). He introduced HTML (HyperText Markup Language), HTTP (HyperText Transfer Protocol), and web browsers, making the Internet easy to use for ordinary people.

  5. 1990s – Public Access

    Internet Service Providers (ISPs) began offering Internet access to the public. Popular web browsers such as Netscape Navigator and Internet Explorer made browsing simple, leading to rapid growth of websites.

  6. 2000s – Broadband and Social Media

    Broadband Internet replaced dial-up connections, allowing faster access to online content. Social media, online shopping, video streaming, and cloud computing became widely used.

  7. Today

    Today, billions of people use the Internet every day. It supports modern technologies such as Cloud Computing, Artificial Intelligence (AI), the Internet of Things (IoT), online education, remote work, digital banking, and many other services.

How the Internet Works

Client-Server Model

Clients (users) send requests, and servers respond with data such as web pages, files, or services.

HTTP / HTTPS

HTTP is the protocol used for communication between browsers and servers. HTTPS adds security via encryption.

DNS-Domain Name Server

DNS converts domain names (like google.com) into IP addresses that computers understand.

What is DNS and Why It is Important?

DNS (Domain Name System) is like the phonebook of the Internet. It translates human-readable domain names into IP addresses so computers can locate websites.

Importance of DNS:
  • Makes websites easy to access
  • Eliminates need to remember IP addresses
  • Improves browsing speed and efficiency

About This Website

This website is developed as part of the Institute of Computer Engineering Technology (ICET) HTML Fundamentals assignment. It serves as a reference guide to understand the basic concepts of the Internet and HTML.

The main purpose of this project is to demonstrate both theoretical knowledge and practical skills in HTML development. It includes explanations of Internet concepts such as how the Internet works, DNS, HTTP, web browsers, and Internet services.

In addition, the website provides a detailed reference of more than 75 HTML tags with examples, helping students understand how each tag is used in real web development.

This project follows semantic HTML5 structure and uses Bootstrap for responsive design to ensure a clean, user-friendly, and professional appearance.

Objective

To understand Internet concepts and HTML fundamentals clearly.

Learning

Improve HTML skills and semantic web development practices.

Outcome

Build a fully structured educational reference website.
